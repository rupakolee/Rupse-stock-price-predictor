# 📊 Rupse Stock ML Predictor (Basic Version)

A simple machine learning project that predicts whether a stock price will go up or down using historical data.

---

## 🚀 Features

- Fetch stock data using Yahoo Finance
- Feature engineering (moving averages, volatility, returns)
- Random Forest classification model
- Basic trading strategy backtest
- Time-series safe train/test split

---

## 🧠 Model

Predicts:
> Will the stock go UP tomorrow?
